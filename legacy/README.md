# Legacy Protocol Snapshots

This folder contains pinned, immutable snapshots of prior protocol files for reference
and auditability.

## Contents

- `rup-protocol-v3.0.yaml`: v3.0.0 snapshot (superseded by the canonical `rup-protocol.yaml`)
- `rup-protocol-v2.1.yaml`: v2.1.0 snapshot

## Policy

- Do not edit files in this folder.
- If a new protocol version is released, add a new snapshot file here and update the
  root `README.md` "Legacy Versions" list.
